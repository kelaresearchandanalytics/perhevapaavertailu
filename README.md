# perhevapaavertailu

This repository will host the source code for [perhevapaavertailu.kela.fi](https://perhevapaavertailu.kela.fi/lang=en) app for calculating family leave benefits.

If you want notifications from this repository, log in and click **Watch**, and you will be reminded when the source code is available. This will take place in spring/summer 2024.
